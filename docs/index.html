LightEditorVidz

Desktop video editor

Create scenes on a canvas with text, images, videos, audio and shapes, animate them, and export a real .mp4 — all locally. Free, open source, no cloud, no watermark.

Download latest release View on GitHub

Install

One-line install (macOS & Linux)

curl -fsSL https://sindus.github.io/light-editor-vidz/install.sh | sh
  • macOS — installs via a personal Homebrew tap and pulls in ffmpeg as a dependency.
  • Linux — downloads the latest AppImage into ~/.local/bin and installs ffmpeg via your package manager if missing.

Manual install

macOS: brew install sindus/light-editor-vidz/light-editor-vidz
Linux/macOS: grab the .deb, .AppImage or .dmg from the releases page.

Uninstall

curl -fsSL https://sindus.github.io/light-editor-vidz/uninstall.sh | sh

Features

Multi-scene timelineA real playback clock, not a fixed-interval hack.
Text · Image · Video · Audio · ShapesDrag, resize and rotate with one shared interaction model.
Composable animationsFade, slide, zoom, rotate, blur, bounce — layered, not overwritten.
Real mp4 exportNative rasterization + ffmpeg, with audio mixing. Not a DOM replay.
Undo/redoPlus duplicate, split, delete, and legacy JSON project import.
Signed auto-updateChecks GitHub Releases, downloads, installs, relaunches.
